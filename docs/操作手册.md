# 情绪擂台 - 操作手册

## 🎮 游戏简介

**情绪擂台**是一款16-bit像素风格的情绪调节小游戏。通过"战斗"的方式，帮助用户以一种轻松、有趣的方式面对和处理负面情绪。

> "面对情绪，不是逃避，而是勇敢地与它对话"

---

## 📱 如何开始

### 1. 启动游戏
- 打开游戏后，点击 **"🎮 开始游戏"** 进入角色选择界面

### 2. 选择角色
- 在"选择你的角色"区域，点击Tab切换 **男性角色** 或 **女性角色**
- 共有6个角色可选：
  | 男性角色 | 女性角色 |
  |---------|---------|
  | 活泼少年（儿童） | 可爱女孩（儿童） |
  | 学生（青少年） | 女学生（青少年） |
  | 上班族（成年） | 职场女性（成年） |

### 3. 给角色命名（可选）
- 在角色选择下方的输入框中：
  - **角色昵称**：为你的角色起一个名字
  - **情绪名称**：为对手情绪起一个名字（如"暴躁的老师"）

### 4. 选择对手情绪
游戏提供6种负面情绪作为对手，难度从低到高：

| 情绪 | 难度 | 时间限制 | 点击次数 |
|------|------|---------|---------|
| 沮丧 | 入门 | 8秒 | 15次 |
| 羞耻 | 简单 | 10秒 | 12次 |
| 恐惧 | 中等 | 6秒 | 18次 |
| 愤怒 | 中等 | 6秒 | 20次 |
| 嫉妒 | 较高 | 5秒 | 22次 |
| 焦虑 | 较高 | 5秒 | 25次 |

### 5. 开始战斗
点击 **"⚔️ 开始战斗"** 进入战斗界面

## ⚔️ 战斗操作

### 战斗界面布局
```
┌─────────────────────────────────────┐
│  [玩家血条]          [情绪血条]      │
│                                     │
│     👤  ←──────距离──────→  😠     │
│                                     │
│  进度: 5/15    用时: 3.2秒  距离: 80px │
│                                     │
│  [点击攻击]  [大招]  [逃跑]          │
└─────────────────────────────────────┘
```

### 操作方式

#### 📱 手机端
- **点击攻击**：点击屏幕下方的 "👆 点击攻击" 按钮
- **使用大招**：点击 "⚡ 大招" 按钮
- **逃跑**：点击 "🏃 逃跑" 按钮

#### 🖥️ 电脑端
- **点击攻击**：点击按钮或按键盘 **空格键/回车键**
- **使用大招**：点击按钮或按键盘 **U键**
- **逃跑**：点击按钮或按键盘 **ESC键**

### 战斗机制

#### 基本攻击
- 每次点击让角色向前冲锋
- 当角色接近情绪时触发攻击
- 每次攻击造成 **8-12点伤害**
- 攻击后角色后退，情绪被打退

#### 大招（20连击）
- 点击大招触发 **20连击爆炸攻击**
- 每次连击造成 **10-15点伤害**
- 总伤害约 **200-300点**，足以击败大部分情绪
- 伴随华丽的爆炸特效和屏幕震动

#### 战斗对话
- 角色被攻击时会显示鼓励话语（如"💪 加油反击！")
- 情绪被攻击时会显示吐槽（如"😱 好强！")

## 🏆 战斗结果

### 胜利 ✅
- 在时间限制内完成所需点击次数
- 情绪血量归零

### 失败 ❌
- 时间耗尽未完成点击次数
- 被情绪攻击过多导致血量归零

### 逃跑 🏃
- 主动选择逃跑
- 不算胜利也不算失败

## 💭 记录心得

战斗结束后，可以记录本次体验：

### 1. 选择标签
点击以下标签之一：
- **💼 工作** - 工作相关的情绪
- **👨‍👩‍👧 家庭** - 家庭相关的情绪
- **❤️ 情感** - 感情相关的情绪
- **🎲 纯倒霉** - 其他原因

### 2. 输入心得
在文本框中输入本次战斗的感受、思考或心情

### 3. 保存
点击 **"💾 保存心得"**，按钮会变成 **"✅ 已保存"**

## 🌬️ 呼吸练习

战斗结束后可选择进入呼吸练习：

### 4-7-8 呼吸法
- **吸气 4秒** - 圆圈放大
- **屏息 7秒** - 圆圈保持脉动
- **呼气 8秒** - 圆圈缩小
- 共进行 **8轮**

### 操作
- 跟随圆圈的节奏呼吸
- 点击 **"✓ 结束练习"** 返回首页

## 📅 战斗日历

点击首页 **"📅 战斗日历"** 查看：

### 日历视图
- 查看某月所有战斗记录
- 不同颜色标记战斗结果：
  - 🟢 绿色：当日全胜
  - 🔴 红色：当日有失败
  - 🟡 黄色：当日胜负混合

### 日期详情
- 点击某天查看当日战斗记录
- 每条战斗右侧显示关联的心得记录

### 翻页操作
- 点击 **"◀ 上月"** / **"下月 ▶"** 切换月份

## ⚙️ 游戏设置

## 🎯 游戏技巧

### 新手建议
1. 从 **沮丧（入门）** 开始，熟悉操作
2. 不要疯狂点击，注意节奏
3. 情绪靠近时再攻击效果更好

### 进阶技巧
1. 利用大招快速击败高难度情绪
2. 记录心得帮助反思情绪来源
3. 坚持每日练习，养成情绪管理习惯

### 注意事项
⚠️ 本游戏仅供娱乐和轻度情绪调节
⚠️ 如有严重情绪问题，请寻求专业心理咨询帮助
